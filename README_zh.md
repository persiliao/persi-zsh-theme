# Persi Zsh Theme

[English](/README.md "English")

## 依赖

- zsh
- [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh "oh-my-zsh")

## 安装 

```sh
$ git clone https://github.com/persiliao/persi-zsh-theme.git
$ cd persi-zsh-theme
# 部署到 oh-my-zsh
$ ./deploy.zsh
$ source ~/.zshrc
```
