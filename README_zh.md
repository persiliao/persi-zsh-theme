## 特性 [可选]

- 添加插件 zsh_reload wd git gitignore git-flow git-flow-avh docker npm node golang wp-cli composer yarn
- Homebrew 使用清华镜像源
- 一些常用的Alias

## 依赖

- zsh
- [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh "oh-my-zsh")

## 安装 

```sh
$ git clone https://github.com/persiliao/persi-zsh-theme.git
$ cd persi-zsh-theme
# 部署到 oh-my-zsh
$ ./install_persi.zsh
$ source ~/.zshrc
```
